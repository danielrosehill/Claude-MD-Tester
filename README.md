# Joke-Claude-Configs

Joke / parody / character-heavy Claude Code configs. Kept deliberately separate from the serious style toolkit (see [Claude-Style-Switcher](https://github.com/danielrosehill/Claude-Style-Switcher)) so the lines between *style* (subtle tonal nudge) and *character piece* (full-on persona for fun) don't blur.

Drop-in fragments for `~/.claude/CLAUDE.md`, output-style files, or wherever you want them.
